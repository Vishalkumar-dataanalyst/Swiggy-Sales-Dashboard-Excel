📊 Swiggy Sales Dashboard - Excel Data Analytics Project

📌 Project Overview

This project is an Interactive Swiggy Sales Dashboard developed in Microsoft Excel to analyze food delivery sales data and generate meaningful business insights.

The dashboard helps stakeholders monitor sales performance, customer ratings, restaurant performance, and regional trends through interactive visualizations and slicers.

🎯 Objectives

- Analyze overall sales performance
- Track order trends and customer ratings
- Identify top-performing restaurants and cities
- Compare Veg and Non-Veg sales
- Analyze state-wise and city-wise sales distribution

🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning & Transformation

📈 Dashboard KPIs

- Total Sales
- Total Orders
- Average Order Value (AOV)
- Average Rating
- Rating Count

📊 Key Insights

- Monthly Sales Trends
- Daily Sales Analysis
- Top 10 Restaurants by Sales
- Top 5 Cities by Revenue
- Regional Sales Distribution
- Veg vs Non-Veg Sales Comparison
- Quarter-wise Performance Analysis

🎛 Interactive Features

Users can dynamically filter the dashboard using:

- Month
- State
- Food Type
- Category

💡 Learning Outcomes

- Data Cleaning and Preparation
- Dashboard Design
- KPI Development
- Data Visualization
- Business Insight Generation
